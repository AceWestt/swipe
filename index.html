Swipe a pure Javascript plugin for touch devices


swipe is a Javascript plugin to be used on touch input devices such as surface, android, iPad, iPhone etc.

Actualy it works on any divice with tuch support and runs Javascript

Features

  • Detects swipes in 2 directions, "left" and "right"
  • Allow to adjest sencitivity
  • Allow to use Event Attrebute "onSwipe"
  • Allow to add addEventListener "Swipe" on any element in Javascript
